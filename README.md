# Kawisara Ph. Portfolio

Personal portfolio website for **Kawisara Ph.**, a Graphic & Web Designer based in Bangkok, Thailand.

## Overview

This project is a responsive single-page portfolio website built with HTML, CSS, and JavaScript using the Bootstrap framework. It showcases professional background, skills, services, and a portfolio of graphic and web design projects.

**URL:** [kawisara-design.com](https://kawisara-design.com)

## Features

-   **Hero Section:** Animated typing effect introduction.
-   **About & Stats:** Professional summary and key statistics.
-   **Skills:** Visual progress bars for technical skills (HTML, CSS, JS, Adobe Suite).
-   **Resume:** Timeline of education and work experience.
-   **Portfolio:** Filterable gallery (Web, Print, e-Books) using Isotope and GLightbox.
-   **Services:** List of design and development services.
-   **Contact:** Contact information and form layout.

## Technologies

-   **HTML5 / CSS3**
-   **Bootstrap 5.3.3**
-   **JavaScript**
-   **Libraries:**
    -   [AOS](https://michalsnik.github.io/aos/) (Animate On Scroll)
    -   [GLightbox](https://biati-digital.github.io/glightbox/) (Lightbox gallery)
    -   [Isotope](https://isotope.metafizzy.co/) (Layout and filtering)
    -   [Swiper](https://swiperjs.com/) (Touch slider)
    -   [Typed.js](https://github.com/mattboldt/typed.js/) (Typewriter effect)
    -   [PureCounter](https://github.com/srexi/purecounterjs) (Counter animation)
    -   [Waypoints](http://imakewebthings.com/waypoints/) (Scroll triggers)

## Setup

1.  Clone the repository or download the files.
2.  Ensure the `assets` folder contains all vendor libraries and custom scripts.
3.  Open `index.html` in a web browser.
    *   *Note:* For full functionality (especially if using AJAX forms or specific JS features), run on a local server (e.g., XAMPP, Live Server).
